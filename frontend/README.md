# Front-End Readme file
### Used Stack
- <b>React</b>
- <b>Typescript</b>


### 페이지 컴포넌트 구성
- <b>login</b> - 로그인 및 회원가입 페이지
- <b>main</b> - 메인페이지 및 이동경로 설정
- <b>reservation</b> - 음식점 예약 페이지
- <b>route</b> - 이동경로 페이지 및 맛집 표시
- <b>storeDetail</b> - 음식점 상세 페이지
    - <b>rates</b> - 음식점 리뷰 컴포넌트
- <b>user</b> - 회원 정보 상세 및 수정 페이지
